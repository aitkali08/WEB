CSS Positioning

В CSS свойство position управляет расположением элементов на странице. Ниже представлены основные типы позиционирования:

Static

Static Box

Это значение по умолчанию. Элемент отображается в нормальном потоке страницы, без смещений.

Relative

Relative Box

Элемент остаётся в потоке, но может быть сдвинут с помощью top, left, right, bottom.

Absolute

Absolute Box

Элемент удаляется из потока и позиционируется относительно ближайшего предка с position: relative.
